# stylized-landscape-env
A performance-optimized, low-poly environment designed for real-time applications and game engines.

## Project Overview
This project focuses on **Environmental Architecture** and **Asset Optimization**. As a finalist IT student, I built this with a focus on clean topology that can run smoothly on mobile devices and low-end hardware.

## Technical Specification 
- **Formats:** .blend, .fbx, .glb
- **Style:** Stylized / Low-Poly
- **Software:** Blender 3D
